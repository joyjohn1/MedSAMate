## 🌟 项目简介

**MedSAMate** 是一款面向医学科研人员、临床医生和医学生的开源半自动医学影像分割软件。我们创新性地融合了 **SAM (Segment Anything Model) 大模型** 与 **交互式 ROI 引导策略**，实现了从"人工粗定位"到"智能细分割"的完美闭环。

## 💡 核心创新

- 🔬 混合智能架构：交互式 ROI 引导 + SAM 大模型微调
- ⚡ 极致效率：单例分割仅需 3 分钟，较手动分割提升 30 倍
- 🎯 精准分割：3D Dice 系数达 0.97，较 U-Net 提升 16.9%
- 💻 轻量化部署：安装包 <100MB，纯 CPU 即可流畅运行

## 🚀 核心特性

##### 🖼️ 多模态影像支持

- ✅ DICOM、IM0、SEG、STL 等主流医学影像格式
- ✅ CT、MRI、dMRI、CBCT 等多模态兼容
- ✅ 自动识别影像参数（体素大小、窗宽窗位）

##### 🔍 智能分割引擎

- ✅ 交互式 ROI 引导：仅需 3-5 个关键切片角点标注
- ✅ 线性插值算法：自动生成全序列精准感兴趣区域
- ✅ SAM 大模型微调：医学专项适配，复杂病例精度提升 30%+

##### 📐 专业可视化

- ✅ 多平面重建 (MPR)：横断面、矢状面、冠状面三视图同步
- ✅ 体绘制 (Volume Rendering)：基于光线投射算法的 3D 重建
- ✅ 实时交互：缩放、平移、旋转、窗宽窗位调整

##### 🤖 模型配置

| 模型名称              | 适用场景                     | 标记方式 |
| --------------------- | ---------------------------- | -------- |
| **Universal Model**   | 通用医学分割（牙齿、心脏等） | 点标记   |
| **MRI lungseg Model** | 肺部专用分割                 | 框标记   |
| **Custom Model**      | 用户自定义模型               | 可配置   |

##### 💾 结果导出

- ✅ NIfTI (.nii.gz)：标准医学影像格式
- ✅ SEG：分割掩膜文件
- ✅ STL：三维模型文件（支持 3D 打印）

## 📖 快速开始

##### 1️⃣ 启动软件

双击桌面快捷方式或运行：

```bash
python main.py
```

##### 2️⃣ 加载影像数据

1. 点击界面左上角 "导入" 按钮
2. 选择影像格式（DICOM / IM0 / SEG / STL）
3. 浏览并选中需要加载的影像文件或文件夹
4. 点击 "确定" 完成加载

##### 3️⃣ 选择分割模型

1. 点击 "Load Models" 打开模型选择对话框
2. 根据分割目标选择合适模型：
   - Universal Model：通用医学分割（牙齿、心脏等）
   - MRI lungseg Model：肺部专用分割
   - Custom Model：用户自定义模型
3. 点击 "确定" 加载模型

##### 4️⃣ ROI 标注与分割

通用模型（点标记）：

1. 选择 "Universal Model"
2. 采用 "点标记" 方式精准定位目标区域
3. 点击 "开始分割" 按钮
4. 查看叠加显示的分割结果

肺部专用模型（框标记）：

1. 选择 "MRI lungseg Model"
2. 采用 "框标记" 方式框选肺部区域
3. 点击 "开始分割" 按钮
4. 系统自动完成肺部区域分割

##### 5️⃣ 保存结果

点击 "Save Results" 按钮，支持导出：

- NIfTI (.nii.gz)：标准医学影像格式
- SEG：分割掩膜文件
- STL：三维模型文件
